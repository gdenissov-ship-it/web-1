## Выполненные пункты задания

1. Создан проект с корректной структурой файлов: `index.html`, `css/styles.css`, папка `img/` с ≥5 изображениями.  
2. Подключен CSS (`<link rel="stylesheet" href="css/styles.css">`).  
3. Собран семантический каркас: `<header>` → `<nav>` → `<main>` (секции `about`, `services`, `team`, `contact`) → `<aside>` → `<footer>`.  
4. Заголовки: 1 × `<h1>` (header), `<h2>` для секций, `<h3>` при необходимости.  
5. Навигация-якоря в `<nav>` с ссылками на секции и подвал.  
6. Таблица с ценами на услуги (`<table>` с `<caption>`, `<th scope="col">`, `<th scope="row">`).  
7. Три вида списков:  
   - `<ul>` – услуги  
   - `<ol>` – команда  
   - `<dl>` – отзывы клиентов  
8. ≥5 изображений с правильными атрибутами `alt`, `width`, `height`; один блок `<figure>` с `<figcaption>`.  
9. Каркас формы с двумя `<fieldset>` и `<legend>`, все поля связаны с `<label for>`.  
10. Типы и валидаторы формы:  
    - `required` на name, email, qty  
    - `type="email"`, `type="number"`, `min="1"`, `max="20"`, `pattern` для телефона, `minlength` для комментария  
11. Минимальные стили для читабельности, видимый фокус элементов формы и ссылок.  
12. Валидированная страница через [W3C Markup Validator](https://validator.w3.org/) – ошибок нет (проверка выполнена: **09.12.2025, 18:00**).  
13. Клавиатурная доступность: навигация Tab/Shift+Tab по меню, таблице и форме работает корректно.
